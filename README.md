# ✅ Tasks Manager App – Powered by Firebase 🚀

**Tasks Manager** is a modern, secure, and intuitive **task management app** built with **Flutter**, Firebase Authentication, and Firestore. Whether you're juggling school, work, or personal goals, this app helps you stay organized while keeping your data safe and personalized.

---

## ✨ Features

- 🔐 **Secure Firebase Authentication**  
  Sign up or log in with your email and password using Firebase Auth.

- 📝 **Add & Manage Personalized Tasks**  
  Create tasks with custom **titles, descriptions, and colors** for easy categorization and visibility.

- 🔍 **User-Specific Task Filtering**  
  Every user sees **only their tasks**, securely stored and queried via Firestore.

- 🔄 **Switch Accounts Effortlessly**  
  Log out and switch between accounts anytime – perfect for shared devices or multiple user profiles.


---

## 🛠️ Tech Stack

| Technology         | Description                                      |
|--------------------|--------------------------------------------------|
| **Flutter**        | Frontend framework for UI                        |
| **Dart**           | Programming language for Flutter                 |
| **Firebase Auth**  | User authentication (email/password)             |
| **Firebase Firestore** | Secure NoSQL cloud database for storing tasks |
